HTML

HTML

Es un lenguage de marcas que sirve esencialmente para la elaboración de páginas web. Es un estándar a cargo del World Wide Web Consortium (W3C) o Consorcio WWW, organización dedicada a la estandarización de casi todas las tecnologías ligadas a la web, sobre todo en lo referente a su escritura e interpretación. Se considera el lenguaje web más importante siendo su invención crucial en la aparición, desarrollo y expansión de la World Wide Web (WWW).

<!DOCTYPE html>

Se pone siempre al inicio de las páginas y sirve para indicar que se está usando HTML.

<html>

Se pone dentro de este tag todo el documento que debe contener la página.

<head></head>

Delimita el encabezado del documento. En el head se introducen los metadatos como, por ejemplo, <meta charset="UTF-8"> sirve para poner la página en UTF-8

<title></title>

Título del documento (se muestra en la pestaña del navegador).

<body></body>

Delimita cuerpo de la página web, en el se incluyen la mayoria de los elementos.

<p></p>

Este elemento se usa para escribir párrafos.

<hn></hn>

Este elemento se usa para poner cabeceras, h1 debería ponerse sólo uno y del resto todos los que se deseen. Cuanto mayor sea el número más pequeño será el tamaño de letra de la cabecera. El máximo es el 6

<a href="enlace" target="_blank"></a>

Este elemento se usa para crear enlaces a otras páginas web lo que escribas dentro del tag saldrá en azul y al hacer click en él te enviará a la dirección a la que hagas referencia dentro de "enlace". target="_blank" es opcional y sirve para que se abra en una nueva pestaña.

<img src="ruta de la imágen" alt="DescDeImagen" width="x" height="y">

Sirve para poner imágenes en la ruta especificada (no valen absolutas). Alt debe incluir una descripción de la imágen para, por ejemplo, las personas conproblemas de visión que el navegador les lee lo que aparece en el alt. Width y height especifican las dimensiones de la misma.

<button></button>

El texto introducido dentro se mostrará como un botón que podrá ser programado.

<ul></ul> <ol></ol> <li></li>

UL y OL sirven para crear una lista, ordenada en el caso de OL, sin ordenar si es UL. LI se usa para especificar cada elemento de la lista. style="list-style-type:disc;" sirve para cambiar la numeración de la lista.

Lista no ordenada:

  • Manzanas
  • Peras
  • Galletas
  • Pan

Lista ordenada:

  1. Pescado
  2. Carne
  3. Ensalada
  4. Agua

<br>

Este elemento sirve para hacer un...
salto de linea.

<lang ="sp-ES">

Se usa para decirle al navegador el idioma y el dialecto que se está usando.

<elemento title="Im a tooltip">

Sirve para indicar ayudas en elementos, cuando pasas el ratón por encima salta el tooltip como en este icono.

<hr>

Se usa para poner una línea como la usada en esta página.

<pre></pre>

Crea un contenedor de texto tal y como tú lo escribes, respeta los espacios, las tabulaciones... 

		___________________________________________
		 Hola, soy un pre y      tengo espacios ;)
		*****************************************

Comentarios

<!-- Comentario --> Sirven para dejar comentarios en el archivo HTML, no se muestran en la web pero se debe tener en cuenta que pueden leerse por cualquier usuario inspeccionando el código fuente de la página.

WorkMaps

Sirven para poder hacer click en cualquier lugar de la imágen para hacer aciones. Se le asigna a la imágen: usemap="#workmap" y con <map name="workmap"> se crean coordenadas:

   Ejemplo:
	<img src="workplace.jpg" alt="Workplace" usemap="#workmap">

	<map name="workmap">
		<area shape="rect" coords="34,44,270,350" alt="Computer" href="">
		<rea shape="rect" coords="290,172,333,250" alt="Phone" href="">
		<area shape="circle" coords="337,300,44" alt="Coffee" href="">
	</map>

Al hacer click en los distintos objetos de la imágen te llevaran a uno u otro link.

Tablas

Vienen muy bien para mostrar datos:
<table> Para abrir/cerrar la tabla.
<tr> Para abrir/cerrar la fila.
<td> Para los elementos generales de cada columna.
<th> Para los elementos de la primera fila o cabecera de la tabla.

<div></div>

Los bloques sirven para absolutamente todo en HTML.
Usado en conjunto con CSS el div sirve para dar estilos a bloques de contenido.

span define una seccion dentro de una linea.

float: left, right... sirve para recolocar el elemento en una capa distinta

<Iframes>

Sirven para introducir páginas dentro de una web.

Formularios:

<form></form> sirve para abrir el formulario dentro del cual se encontrarán los siguientes elementos: name sirve para poner nombre al campo, es importante en lo botones para que se marque sólo una opción. value sirve para trabajar con el formulario. action indica una ruta donde irá la info de dicho formulario. method para seleccionar el modo de envio del formulario get es malo pero lo usaremos.

<select></select> sirve para introducir dentro opciones y asi tener una lista para seleccionar una de esas opciones.<option value="audi">Audi</option>

Tipos:

  • <input type="text"> sirve para que el usuario introduzca texto.
  • <textarea></textarea> sirve para que el usuario introduzca más lineas.
  • <input type="email"> sirve para que HTML5 valide que se trata de un eMail.
  • <input type="submit"> se usa como botón de aceptar, para enviar el formulario.
  • <input type="password"> para pedir contraseñas.
  • <input type="radio"> para poner botoncitos redonos. Importante el nombre.
